## 2. 核心功能

### 2.1 记录开销

用户可记录一笔消费，包含以下信息：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 金额 | 数字 | 是 | 支持两位小数，上限 9,999,999 |
| 分类 | 选择 | 是 | 从预设9大类中选择 |
| 子类 | 选择 | 是 | 选择大类后，从对应子类中选择 |
| 日期 | 日期 | 是 | 默认为今天，可修改 |
| 备注 | 文本 | 否 | 额外说明文字，如"中午请客户吃饭" |

**操作流程：**
点击底部 "+" 按钮 → 弹出全屏录入页 → 使用数字键盘输入金额 → 选分类 → 选子类 → 修改日期（可选） → 写备注（可选） → 点击"保存"

#### 娱乐 🎮

#### 其他 📦

共 **9大类、34子类**。

### 2.3 查看账单

- 首页显示当月所有开销列表，按日期倒序排列
- 每条记录显示：分类图标 + 子类名称 + 金额 + 日期
- 今日/昨天记录显示"今天"/"昨天"标签
- 支持按月份切换（上个月/下个月）
- 支持左滑删除记录（带确认弹窗）
- 当月无记录时显示空状态提示

### 2.4 月度汇总

首页顶部展示当月汇总信息：
- 当月总支出
- 当月总笔数
- 日均支出（按有消费的天数计算）

### 2.5 预算管理

- 用户可在"设置"页设置每月预算金额
- 首页显示预算进度条（已花费/总预算）
- 进度条颜色分三档：
  - 橙色（正常）：0% ~ 79%
  - 黄色（警告）：80% ~ 99%
  - 红色（超支）：100%+

### 2.6 统计图表

- **月度总览卡片**：总支出、日均支出、消费天数
- **分类饼图**：展示各大类花费占比（环形图）
  - 带图例显示分类名称、比例、金额
  - hover 显示详细信息
- **每日趋势折线图**：当月每日花费变化趋势
  - X轴为日期，Y轴为金额
- 支持按月份切换查看

### 2.7 设置

- **预算设置**：修改每月预算金额
- **数据导出**：导出为 CSV 文件（可用 Excel 打开）
- **清空数据**：删除所有记录（需二次确认）
- **版本信息**：显示应用版本号

## 3. 非功能性需求

| 需求 | 说明 |
|------|------|
| 离线可用 | 完全离线运行，无需网络 |
| 数据安全 | 数据仅存储在用户手机本地，不上传任何服务器 |
| 性能 | 列表渲染流畅，页面切换无卡顿 |
| 兼容性 | 适配 iPhone SE ~ iPhone Pro Max 全系屏幕 |
| 安装 | 通过 Safari"添加到主屏幕"使用，类原生体验 |
| 无广告 | 整个应用无任何广告和商业推广 |
