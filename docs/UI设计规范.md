# DK日常开销 —— UI/UX 设计规范

## 1. 设计原则

- **简洁**：界面干净，信息层级清晰，无冗余元素
- **直观**：用户一看就懂，零学习成本
- **易用**：大字、大按钮，方便单手操作
- **一致**：全应用统一的设计语言

---

## 2. 色彩系统

### 主色调

| 用途 | 色值 | 色块 |
|------|------|------|
| 主色 | `#FF6B35` | 🟠 |
| 主色-亮 | `#FF8C42` | 🟡 |
| 主色-暗 | `#E55A2B` | 🟤 |
| 主色-背景 | `#FFF3ED` | 浅橙 |

### 功能色

| 用途 | 色值 |
|------|------|
| 危险/删除 | `#FF3B30` |
| 成功 | `#34C759` |
| 警告 | `#FF9500` |

### 中性色

| 用途 | 色值 | 说明 |
|------|------|------|
| 文字主色 | `#1C1C1E` | 标题、金额 |
| 文字次要 | `#6C6C70` | 说明文字 |
| 文字浅色 | `#AEAEB2` | 占位符、辅助文字 |
| 背景色 | `#F5F5F7` | 页面背景 |
| 卡片色 | `#FFFFFF` | 卡片背景 |
| 边框 | `#E5E5EA` | 分割线、边框 |

---

## 3. 布局规范

- **最大宽度**：430px（适配 iPhone 最大宽度）
- **页面内边距**：左右 16px
- **圆角规范**：大卡片 16px，小元素 10px
- **卡片阴影**：`0 2px 12px rgba(0,0,0,0.06)`

### 安全区域

- 底部适配 `safe-area-inset-bottom`
- 内容在安全区域内展示

---

## 4. 字体规范

- 字体栈：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Helvetica Neue"`
- 金额为大号字（36px~48px），粗体
- 正文 15px~17px
- 辅助文字 11px~14px

---

## 5. 组件规范

### 5.1 底部导航栏

- 三个 Tab：账单、统计、设置
- 中间为突出的圆形"+"按钮（52x52px）
- 选中状态为主色，未选中为浅灰色

### 5.2 记录卡片

- 左侧42x42px圆角图标（浅橙色底）
- 中间：子类名称（15px）+ 备注（12px浅色）
- 右侧：金额（16px粗体）
- 左滑露出红色"删除"按钮

### 5.3 数字键盘

- 3列网格布局
- 56px高按键，12px圆角
- 包含 0-9、小数点、删除键、完成键
- "完成"按钮为主色填充

### 5.4 分类选择

- 5列网格（小屏自动变为4列）
- 选中时：橙色边框 + 浅橙背景
- 子类为胶囊形按钮横向排列

### 5.5 预算进度条

- 8px 高度，圆角
- 渐变填充：亮橙 → 主橙
- 三档颜色变化（正常/警告/超支）

---

## 6. 交互动效

| 场景 | 动效 | 时长 |
|------|------|------|
| 新增页弹出 | 从底部滑入 (slideUp) | 300ms |
| 按钮点击 | 缩放 0.95 | 150ms |
| Toast 提示 | 淡入 + 弹性放大 | 250ms |
| 弹窗 | 背景淡入 + 内容弹性缩放 | 250ms |
| 页面切换 | 即时切换 | 0ms |
| 删除滑动 | 平滑位移 | 250ms |
